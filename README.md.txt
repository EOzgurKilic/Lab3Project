This is a simple front-end video gallery project created using **HTML**, **CSS**, and **JavaScript**. It showcases a collection of four videos with thumbnails, a main landing page, and individual video pages with watch/download features.

---

#### 1. **Install Live Server Extension**

- Open **Visual Studio Code**
- Go to the **Extensions** panel (Ctrl + Shift + X)
- Search for `Live Server` by **Ritwick Dey**
- Click **Install**

#### 2. **Run the Project**

- Open the `Lab3Project` folder in VSCode
- Right-click `index.html` in the Explorer sidebar
- Click **"Open with Live Server"**

This will start the server and open the project in your default browser at a local address like:
http://127.0.0.1:5500/index.html


Now everything including **video downloads** will work as expected.
